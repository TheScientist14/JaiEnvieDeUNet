Quand on start le lobby, c'est normal si on doit attendre 3/4 minutes la première fois, le temps que le premier serveur soit alloué,
car avoir des serveurs qui run tout le temps force un coût que nous ne souhaitions pas avoir à supporter.

FFA et TeamDeathmatch ne fonctionnent pas, nous avons préférés nous focus sur comment faire marcher via serveur dédié au lieu d'avoir le host qui lui-même héberge le serveur.
Ça a été codé, mais ça ne fonctionne pas par manque de temps.

Les teams sont implémentées, pas d'UI dessus, les teams sont aléatoires.

On a un seul type d'ennemis (Hoverbot) qui est l'ennemi à distance.

Nous avons utilisés les services d'Unity suivant:
- Lobby
- Matchmaker
- Game Server Hosting

le chemin Main Menu -> Scène PVE est censé fonctionner
